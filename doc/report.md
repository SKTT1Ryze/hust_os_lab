## 实验目的
+ 掌握 Linux 下共享内存的概念与使用方法
+ 掌握环形缓冲的结构与使用方法
+ 掌握 Linux 下进程同步与通信的主要机制

## 实验内容
+ 利用多个共享内存构成的环形缓冲，将源文件复制到目标文件，实现两个进程的眷抄
+ 对下列参数设置不同的取值，统计程序并发执行的个体和总体执行时间，分析不同设置对缓冲结果和进程并发执行的性能影响，并分析其原因：信号灯的设置，缓冲区的个数和进程执行的相对速度
